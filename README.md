## Installation
#### From PyPI:
```console
pip install TP-VulnBox
```
#### From Source:
```console
git clone https://github.com/truocphan/TP-VulnBox.git --branch <Branch/Tag>
cd TP-VulnBox
python setup.py build
python setup.py install
```
